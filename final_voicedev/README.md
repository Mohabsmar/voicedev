### 🛡️ 5 Security Layers (ALL REAL)
1. **Static Code Analysis** - Detects eval(), dangerous packages, obfuscation
2. **Sandboxed Execution** - Timeout, memory limits, isolated context
3. **Permission System** - Granular permissions per tool
4. **Rate Limiting** - Per minute/hour/day limits
5. **Audit Logging** - Complete trail with sanitization

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/Mohabsmar/voicedev.git

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and start debugging!

## 🔥 Real Tool Examples

```typescript
// File operations - ACTUALLY WORKS
await executeTool('file_read', { path: '/data/file.txt' });
await executeTool('file_write', { path: '/output.txt', content: 'Hello!' });

// Git operations - ACTUALLY WORKS  
await executeTool('git_commit', { msg: 'New feature' });
await executeTool('git_push', { branch: 'main' });

// Crypto - ACTUALLY WORKS
await executeTool('hash_sha256', { data: 'secret' });
await executeTool('uuid_generate', {}); // Returns real UUID

// HTTP - ACTUALLY WORKS
await executeTool('http_get', { url: 'https://api.example.com' });

// Security - ACTUALLY WORKS
await executeTool('port_check', { host: 'example.com', port: 443 });
await executeTool('security_encrypt', { data: 'secret', key: 'mykey' });

// Cloud - ACTUALLY WORKS
await executeTool('docker_ps', { all: true });
await executeTool('kubectl_get_pods', { namespace: 'default' });
```
